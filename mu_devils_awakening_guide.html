Atributos Elementais
Dano, Defesa, Crítico, Absorção e Resistência de Elemento
Atributo Para que serve
Element ATKDefine o dano elemental base causado aos inimigos.
Element DEFReduz o dano elemental recebido.
Element ResistanceAumenta a resistência contra ataques elementais.
Element PenetrationReduz a resistência elemental do alvo, aumentando o dano causado.
Element DEF IgnoringIgnora parte da defesa ou resistência elemental do inimigo.
Element Crit ChanceAumenta a chance de causar crítico elemental.
Element Crit ResistReduz a chance de sofrer críticos elementais.
Element Crit BonusAumenta o dano causado pelos críticos elementais.
Element Crit ReductionReduz o dano dos críticos elementais recebidos.
Element Lucky StrikeAumenta a chance de ativar ataques especiais (Lucky Strike).
Element Lucky ResistReduz a chance de sofrer Lucky Strike.
Element DMG AdditionAdiciona uma quantidade fixa de dano elemental.
Element DMG BonusAumenta percentualmente o dano elemental causado.
Element DEF Ignoring (2)Chance de ignorar parte da defesa elemental do alvo.
Elemental BreakdownIgnora ou reduz a resistência elemental do inimigo, aumentando o dano efetivo.
Damage AmplificationMultiplica o dano final causado após os demais cálculos.
Damage Bonus AuxAumenta a eficiência de bônus de dano provenientes de sistemas auxiliares.
Element DMG ResReduz o dano elemental recebido.
Element Fury (ATK)Aumenta a eficácia dos ataques Fury Strike.
Element Fury (DEF)Reduz o efeito dos Fury Strike recebidos.
Element Absorb REDReduz a absorção de dano de elemento do inimigo.
HP BonusAumenta a vida máxima do personagem.
Elemental DEFAumenta a defesa elemental geral.
Elemental RageAumenta o dano elemental causado.
Elemental AbsorbReduz a absorção elemental do alvo.
Element Crit RESReduz os efeitos dos críticos elementais recebidos.
